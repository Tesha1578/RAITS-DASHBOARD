# Driver Link Pro

India's first AI-powered driver network, offering professional, on-demand drivers for your personal vehicle. Book a verified professional driver near you — by the hour, one-way, or round trip.

---

## 🔗 Live Application & About

The application is deployed and live on Vercel:

👉 **[https://driver-link-app-nine.vercel.app/](https://driver-link-app-nine.vercel.app/)**

---

## 📱 Features

- **PriceNet AI Engine:** Dynamic fair-fare calculation engine based on hour, distance, traffic, and real-time demand.
- **MatchCore Matching:** Quick match with nearby drivers in under 30 seconds.
- **DemandPulse Forecasting:** Dynamic resource allocation system to forecast demand across hotspots.
- **FraudShield Security:** Real-time route verification, SOS alarms, and driver behavior analysis.
- **Premium User Experience:** Built with a modern, glassmorphic dark interface optimized for mobile views.

## 🛠️ Technology Stack

- **Framework:** React 19 + TypeScript + Vite 7
- **Styling:** Tailwind CSS + Radix UI Primitives
- **Icons:** Lucide React
- **Animations:** Tailwind Animate + Custom Floating Animations

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the local development server:
   ```bash
   npm run dev
   ```

3. Open your browser to:
   ```
   http://localhost:3000/
   ```

4. Build for production:
   ```bash
   npm run build
   ```
